Coasty Check

Smart India Hackathon 2024 - Problem Statement ID: SIH1656

Project Overview

Coasty Check is a mobile application designed to provide recreational suitability information for beach locations across India. The app uses real-time data from INCOIS APIs to provide safety alerts and forecasts for tourists, ensuring a safer beach experience. We also developed the website for the activity manager and beach officer to add the notifications,activities,review of the beach in the app.

Key Features

  • Interactive UI with color-coded safety indicators
  • Real-time alert notifications for beach conditions
  • Nearby amenities information
  • Geospatial visualization of beach safety data
  • Fast & foresighted warning alerts
  • Accurate forecasting using ML models
  • Simple and intuitive user interface
